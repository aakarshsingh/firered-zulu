# 🔥 firered-zulu

> A custom battle planner, team tracker, and run log built while actively playing Pokémon FireRed.  
> Trainer **ZULU** · ID **07533** · 🏆 **Champion of Kanto**

---

## What is this?

A single-file HTML planner I built alongside a FireRed playthrough on the Nintendo Switch (Feb–Mar 2026). It started as a simple level tracker and grew — version by version, fight by fight — into a full tactical reference.

No frameworks. No build step. Just one HTML file you open in a browser.

## Gen 3 things that will surprise you

- **Shadow Ball is Physical.** Snorlax has 110 Atk. This is why he demolished Agatha.
- **Dragon Claw is Special.** Charizard has 109 Sp.Atk. Lucky.
- **Bite is Special.** Dark = Special in Gen 3. Jolteon's post-game upgrade hits with 110 Sp.Atk.
- **Gyarados is nerfed.** 125 Atk but Surf uses 60 Sp.Atk. A different game's problem.

## Usage

```
git clone https://github.com/aakarshsingh/firered-zulu
open firered-planner-v15.html
```

That's it. No server. No install. Checkboxes persist via `sessionStorage`.
